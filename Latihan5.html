Belajar CSS

Apa itu CSS?

CSS adalah salah satu bahasa yang wajib Anda ketahui saat belajar membuat website. Tanpanya, tampilan website akan kurang menarik, dan perlu upaya lebih untuk melakukan perubahan pada elemen tampilan.

Bagaimana menggunakan CSS?

Ada 3 cara menggunakan CSS:

  1. Iniline Styles: Menuliskan kode CSS langsung pada tag yang akan diubah atau atur di style-nya
  2. Internal/Embedded Styles: Menuliskan kode CSS pada bagian header dokumen HTML
  3. Eksternal/Linked Styles: Menuliskan kode CSS pada file terpisah dan dipanggil melalui tag link dari dokumen HTML
